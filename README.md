## Getting Started

1. Clone this repository

    ```bash
    git clone https://github.com/tranphongdev/beautice-clinic-reactjs.git
    ```

2. Go to project directory and install all dependencies

    ```bash
    cd beautice-clinic-reactjs
    yarn # or npm install
    ```

3. Available scripts

    ```bash
    # Run development server
    npm run dev

    # Build project
    npm run build

    # Serve project
    npm run serve
    ```
